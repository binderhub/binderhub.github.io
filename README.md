# binderhub.github.io

Example website:

https://binderhub.github.io/

# Setup

## Python requirements:
```pip install pyyaml```

## Install Jekyll
https://jekyllrb.com/docs/installation/

## Setup GitHub Pages website:
https://pages.github.com/

# Usage
Make site:

```python /make_site.py```

Serve site locally:

```bundle exec jekyll serve```

Update public site:

Just push your code to your GitHub pages repository!
